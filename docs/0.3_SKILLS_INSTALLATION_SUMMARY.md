## üìã Common Questions

**Q: How do I invoke a specific skill?**  
A: In Copilot Chat, type `@skill-name` followed by your question. The skill provides specialized context for that domain.

**Q: Can I use multiple skills in one prompt?**  
A: Yes! Combine them: `@react-dev @react-performance-optimization How do I render 10k points in React Three Fiber efficiently?`

**Q: What if a skill doesn't seem to be working?**  
A: Restart VS Code and Copilot Chat. Skills are loaded on startup.

**Q: Where are global skills stored?**  
A: `~/.agents/skills/` - You can view/edit agent definitions there if needed.

**Q: Can I uninstall a skill?**  
A: Yes: `npx skills remove skill-name`. To remove globally: `npx skills remove skill-name -g`
